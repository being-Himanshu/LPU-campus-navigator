# 🚀 LPU Campus Navigator

An AI-powered Smart Campus Navigation System designed for Lovely Professional University (LPU). This project helps students and visitors easily navigate the campus, locate buildings, explore indoor maps, view 360° panoramas, and monitor IoT sensor data.

---

## 📌 Project Overview

LPU Campus Navigator is a smart navigation platform developed to improve the campus experience. It combines interactive maps, indoor navigation, 360° virtual tours, and IoT-based sensor integration to help users find locations quickly and efficiently.

---

## ✨ Features

- 🗺️ Smart campus navigation
- 🏢 Building location finder
- 📍 Indoor map navigation
- 🌐 360° virtual campus view
- 📡 IoT sensor integration
- 📊 Real-time sensor simulation

## 🚀 How to Run

### Frontend

1. Download or clone the repository.
2. Open `index.html` in your browser.

### Python

```bash
python sensor_server.py
```

### Arduino

Upload the code from:

```
sensor_node/
```

to your Arduino board.
